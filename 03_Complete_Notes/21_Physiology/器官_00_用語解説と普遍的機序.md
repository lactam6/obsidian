- RTK

## 機序
PIP2 -> PI3K -> PIP3 -> PKB
PIP2 -> (Gq -> PLC) -> IP3, DAG
DAG -活性化-> PKC
IP3 -> 小胞体のIP3受容体
GPCR -> Gs -> AC -> cAMP -> PKA

Gi -抑制-> AC
GC -> cGMP -> PKG
